## 2. 推奨構成と費用シミュレーション

### パターンA: 最小構成（1〜2人で試す）

| 項目 | プラン | 月額/人 |
|------|--------|---------|
| Claude | Max 5x | $100（約15,000円） |
| GitHub | Free | $0 |
| **合計** | | **約15,000円/人/月** |

- まず1人で試して効果を検証するのに最適
- GitHub Freeでもプライベートリポジトリは作成可能

### パターンB: 小規模チーム（3〜5人）

| 項目 | プラン | 月額/人 |
|------|--------|---------|
| Claude | Team Standard（年払い） | $20（約3,000円） |
| GitHub | Team | $4（約600円） |
| **合計** | | **約3,600円/人/月** |

- 5人の場合: **約18,000円/月（年間約216,000円）**
- Team Standardは利用量がProと同等のため、ヘビーに使う開発者はPremium席（$100/人/年払い）への変更を検討

### パターンC: 小規模チーム＋ヘビー利用者あり

| 項目 | プラン | 月額/人 |
|------|--------|---------|
| Claude（ヘビー利用者） | Team Premium（年払い） | $100（約15,000円） |
| Claude（一般利用者） | Team Standard（年払い） | $20（約3,000円） |
| GitHub | Team | $4（約600円） |

- 例: Premium 2人 + Standard 3人 + GitHub 5人の場合
- **約51,600円/月（年間約619,200円）**

## 3. 各プランの利用量と開発スタイルの目安

| プラン | Pro比の利用量 | リセット間隔 | 月額（年払い） |
|--------|-------------|-------------|--------------|
| **Team Standard** | **1.25倍** | 週単位 | $20/人 |
| Team Premium | 6.25倍 | 週単位（モデル別上限あり） | $100/人 |
| Max 5x | 5倍 | 5時間ローリング | $100/人（月払いのみ） |
| Max 20x | 20倍 | 5時間ローリング | $200/人（月払いのみ） |

> **出典**: [Claude Team Premium vs Max plans](https://lord.technology/2026/03/28/claude-team-premium-vs-max-plans-usage-limits-pricing-and-which-to-choose.html) / [Claude Code Pricing 2026](https://www.ssdnodes.com/blog/claude-code-pricing-in-2026-every-plan-explained-pro-max-api-teams/)

### Standard（Pro比1.25倍）で十分なケース

- コードレビューの補助、質問への回答
- 1日数回のコード生成・修正依頼
- ドキュメント作成支援
- 軽めのリファクタリング

### Standardでは厳しいケース

- **1日中Claude Codeで対話しながら開発**（すぐ上限に達する）
- 大量のファイルを連続で編集
- 複雑な機能の一括実装
- 長時間のペアプログラミング的なセッション

### 結論

**Standardは「たまにAIに聞く」レベル**の利用量。Claude Codeで本格的に開発するなら **Team Premium（$100/年払い）** または **Max 5x（$100/月払い）** が現実的。

## 4. SVNからの移行について

- 既存SVNを **即座に廃止する必要はない**
- 新規案件のみGitHubで開始し、段階的に移行する方法が低リスク
- `git svn` コマンドでSVNリポジトリをGitに変換することも可能

## 5. セキュリティ上の注意

| 項目 | 対応 |
|------|------|
| 業務コードのClaude送信 | Anthropicはユーザーデータを学習に使用しない（規約確認必須） |
| チーム管理 | Team/Enterpriseプランなら管理者が利用ポリシーを制御可能 |
| GitHub アクセス制御 | Team以上で保護ブランチ・レビュー必須設定が可能 |
| 個人アカウントの業務利用 | 避けるべき。会社契約のアカウントを使用すること |

## 6. Claudeの動作環境について

| | 動作場所 | 説明 |
|--|---------|------|
| **モデル（AI本体）** | Anthropicクラウド | 常にクラウド上で推論を実行 |
| **Claude Code（CLI/IDE）** | ローカルPC | ローカルで動くが、質問のたびにクラウドへ通信 |

- **オフラインでは使用不可**（インターネット接続必須）
- コードや質問内容は**Anthropicのサーバーに送信される**（学習には使用しないとの規約）
- ローカルにモデルをダウンロードして動かすことは**できない**
- 企業導入時は「業務コードが外部に送信される」点についてセキュリティ部門への確認が必要

1. **まず1人でMax 5x（$100/月）+ GitHub Freeで試す**

3. セキュリティ要件が厳しければ **Enterprise** を検討
